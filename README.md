# OpenAI Model Quality Analysis (10,000 Interactions)

**End-to-End Data Analyst Portfolio Project**

Analyzed **10,000 synthetic user interactions** across GPT-3.5-turbo, GPT-4, and GPT-4o.

### Key Results
- **Best Model**: `gpt-4o` — **4.52/5** satisfaction, **4.1%** hallucination rate
- Structured prompts improved satisfaction by **+0.68** and reduced hallucinations by **41%**
- Full pipeline: Data generation → EDA → Statistical tests → Visualization → Business recommendations
